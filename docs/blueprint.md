# **App Name**: Broker Bug Simulator

## Core Features:

- Account Creation (Mock): Simulate the creation of a user account with a provided name.
- Simulated Deposit: Provide a visual simulation of depositing funds into the account, increasing the displayed balance.
- Trade Simulation: Simulate opening a trade with the entire balance, freezing the balance amount.
- Balance Glitch Animation: Run an animation to simulate a bug that increases the account balance from 1,000 to 10,000.
- Hacker Overlay: Display a 'hacker overlay' during the glitch animation to enhance the visual effect of a security breach.
- Action Log (Fictitious): Display a log of simulated actions, such as account creation, deposit, and trade opening.
- Simulation Reset: Allow the user to reset the simulation to its initial state, with a balance of 1,000 and no active overlays.

## Style Guidelines:

- Primary color: Indigo (#4F46E5) to convey trustworthiness and tech.
- Background color: Dark gray (#1A202C) to provide contrast and a sense of sophistication.
- Accent color: Emerald green (#10B981) to highlight positive actions or simulated gains.
- Body and headline font: 'Inter', a sans-serif font for clean readability.
- Code font: 'Source Code Pro' for displaying log data and code snippets.
- Use simple, outline-style icons for interface elements.
- Incorporate smooth transitions and animations for state changes and the 'glitch' effect.
